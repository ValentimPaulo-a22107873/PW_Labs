Html5-Css

In this page you will be able to see what we used to make the site.

Html5 is used to organize the info that is displayed to the users.

Elements Used Description
Header ✔️ Used to create the header for every page.
Nav ✔️ Used to separate the menu that links to the other pages, from the rest.
Main ✔️ Used to organize the main information of the page.
Article ❌ Used to give independent information to the user.
Section ✔️ Used to separate diferent types of information.
Aside ✔️ Used to give extra information.
Footer ✔️ Used to create a footer in every page.

Css is used to style each page of the site, meaning that you will be able to change color, letter types, etc...

Elements Used Description
Universal ✔️ Used to change everything for example the background color or the text type.
Type ✔️ Used to change everything from a certain type.
Class ✔️ Used to change everything from a certain classe.
Id ✔️ Used to change everything from a certain id.
Atribute ✔️ Used to change everything from a certain atribute.
